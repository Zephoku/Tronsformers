Game Instructions
~~~~~~~~~~~~~~~~~
Open "index.html" in a web browser (preferably Firefox or Chrome)
You can also access the game on our site: tessi.me/tron
Allow the web browser to access your webcam
After webcam is set, you may control the camera in 3d environment.
Avoid the bullets!
Use WASD to move the pyramid

Game Features
~~~~~~~~~~~~~
Custom, self-built game engine
Facial detection and tracking using headtrackr.js
Shadows
Collision Detection
Top score tracker
Levels of difficulty
Includes debugger information about webcam status
