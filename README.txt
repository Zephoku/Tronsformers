Game Instructions
~~~~~~~~~~~~~~~~~
open "index.html" in a web browser (preferably Chrome or Firefox)
Allow the web browser to access your webcam
After webcam is set, you may control the camera in 3d environment.
Avoid the bullets!

Game Features
~~~~~~~~~~~~~

Facial detection and tracking
Shadows
Collision Detection
